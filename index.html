Graphical User Interface (GUI)

The PinSight interface is organized into three core modules: (1) Data Import and Pre-processing, (2) Interactive Pin Editing, and (3) Visualization and Reporting.

(a) Annotated full interface. (1) Board dimension inputs for pixel-to-metric scaling; (2) Optional leveling/detrending toggle; (3) Image import; (4) Manual profile module; (5) Folder import; (6) Export module; (7) Batch statistics panel; (8) Interactive pin editing toggle; (9) Results analysis panel.
(b) Interactive editing mode enabling manual correction of detected pins.
(c) Visualization module displaying reconstructed surface profile, autocorrelogram, and semivariogram.
(d) Manual profile acquisition workflow including ROI selection and axis calibration.
