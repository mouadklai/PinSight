PinSight

A computer vision–based software for automated, accurate, and reproducible processing of pin profiler photographs for soil surface roughness analysis.

Current version: v0.1

Download PinSight (Windows)

What PinSight Does

PinSight digitizes mechanical pin profiler measurements from photographs, transforming raw images into quantitative soil surface roughness indicators. The software follows peer-reviewed methodology (Nolin et al., 2005) and integrates automated computer vision with expert-controlled manual correction.

Core Features

Automated Pin Detection

Robust computer vision pipeline for board detection, perspective correction, and pin extraction under field conditions.

Manual Pin Editing

Interactive editor allowing users to add, delete, or reposition pins when automatic detection requires refinement.

Scientific Roughness Metrics

RMS height, correlation length, maximum height, chain index (tortuosity), and fractal dimension computed with optional detrending.

Batch Processing

Process large image sets simultaneously with automatic statistics (mean, standard deviation, min, max).

Advanced Visualization

Microtopography profiles, autocorrelograms, and log–log semivariograms generated automatically for each measurement.

Export & Reproducibility

Export full datasets and batch statistics to CSV for reporting, modeling, and publication.

Application Overview

Full PinSight interface showing image processing, metrics, batch statistics, and visual analysis.

Interactive Pin Editing Mode

When editing mode is enabled, users can interact directly with the detected pins to ensure precise representation of the soil surface profile. All metrics are recalculated instantly after modification.
