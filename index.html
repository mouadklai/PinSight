Graphical User Interface (GUI)

The PinSight interface is designed to be intuitive for first-time users while offering deep control and quality assurance for advanced research applications. The workflow is organized into three core functional modules: (i) Data Import and Pre-processing, (ii) Interactive Pin Editing, and (iii) Visualization and Reporting.

(a) Annotated full interface. (1) Board dimension inputs for pixel-to-metric scaling; (2) Optional leveling/detrending toggle to remove large-scale slopes; (3) Image import button for loading files; (4) Manual profile module for user-guided digitization; (5) Folder import option for batch processing; (6) Export module for generating statistical reports; (7) Batch statistics panel displaying aggregated roughness indicators; (8) Interactive pin editing toggle for manual correction; (9) Results analysis panel to switch between numerical tables and visualizations.
(b) Interactive editing mode enabling manual correction of detected pins.
